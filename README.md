# Code for An Environment-Aware GNSS Dataset

This repository provides the data processing code for the paper **“An Environment-Aware GNSS Dataset”**. The code is used to convert raw u-blox UBX JSON messages into structured GNSS observation files and further construct satellite–time feature matrices for dataset release and technical validation.

## Code Description

- **Process-1: `raw_ubx_to_hourly_json.py`**  
  This script reads raw UBX JSON messages and converts them into hourly processed files, including `observation`, `satelliteInformation`, and `pvtSolution` files.

- **Process-2: `hourly_json_to_aligned_matrices.py`**  
  This script reads the hourly processed files generated by Process-1, performs satellite-position-based cross-cycle alignment, and generates satellite–time feature matrices organized by site, constellation, frequency, and observation feature.

## Notes

The input and output paths in the scripts are set as relative paths. Users can modify the configuration section at the beginning of each script according to their local data structure.
